# Machine Learning Studies
 
Estou aprendendo Machine Learning e uso esse repositório pra guardar os algoritmos que vou implementando ao longo do curso.
 
A ideia é implementar tudo do zero pra entender a matemática por trás! 

Além disso, também estou explorando análise de dados na prática, como correlação entre variáveis e visualização com gráficos, para entender como os dados se comportam antes mesmo de aplicar modelos.

## Algoritmos
 
### Regressão Linear
`algoritmos/regressao-linear.py`
 
Previsão de valores com base em dados históricos. Dado o histórico de horas estudadas e notas de alunos, por exemplo, o modelo consegue prever a nota de quem vai estudar X horas.
 
Implementado com correlação de Pearson, cálculo de inclinação, interceptação e previsão final pela equação `y = ax + b`.

### Correlação entre variáveis (mtcars)
`analise-dados/correlacao_mtcars.py`

Exploração de relações entre variáveis usando um dataset de carros, analisando características como peso, potência, cilindros e consumo.

O código calcula a correlação entre as variáveis, com valores que vão de:

- **1** → variáveis diretamente proporcionais  
- **0** → não existe uma relação clara  
- **-1** → variáveis inversamente proporcionais  

Também gera um mapa de cores (heatmap) e gráficos de dispersão para visualizar essas relações na prática, facilitando a identificação de padrões nos dados.
