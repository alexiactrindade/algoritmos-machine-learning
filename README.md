## Algoritmos
 
### Regressão Linear
`algoritmos/regressao-linear.py`
 
Previsão de valores com base em dados históricos. Dado o histórico de horas estudadas e notas de alunos, por exemplo, o modelo consegue prever a nota de quem vai estudar X horas.
 
Implementado com correlação de Pearson, cálculo de inclinação, interceptação e previsão final pela equação y = ax + b.
 
#### Como executar?
 
```bash
git clone https://github.com/seu-usuario/machine-learning-studies
cd machine-learning-studies
python algoritmos/regressao_linear.py
```
